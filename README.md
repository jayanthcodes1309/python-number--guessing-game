# Number Guessing Game 🎯

A simple Python number guessing game where the player tries to guess a randomly generated number within limited attempts.

## Features
- Random number generation
- Easy / Medium / Hard difficulty levels
- Limited attempts
- Input validation
- Play again option

## How It Works
1. Choose a difficulty level
2. The computer generates a random number
3. Enter guesses in the terminal
4. The game gives hints:
   - Too high
   - Too low
5. Win before attempts run out

## Technologies Used
- Python
- random module
- while loops
- conditional statements
- functions

1. Install Python
2. Download the project
3. Open terminal in the project folder
4. Run:
